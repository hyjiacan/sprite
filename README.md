# css sprite

```

	css雪碧图简单制作工具
	
	可以通过图片，直接生成sprite文件，并且生成代码
	
	可以通过鼠标点击调整图片位置

	可以添加单张图片，以及删除单张图片

	可以保存为.sprite文件，以后好维护
	
```
